### 赛题解读：
	数据：执御平台上沙特阿拉伯市场的历史数据积累，数据时间跨度为2017年3月1日至2018年3月16日数据。
  
	商品属性数据，商品的市场表现数据，营销信息数据，也可爬取外部数据如：宗教和天气数据。
  
	目标：预测2018年5月1日起，未来5周的sku_id的每一周的销量值。
